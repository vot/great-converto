API

This API provides three endpoints.

You can POST multipart forms or JSON data to create a job.

Throttling may apply.

POST multipart/form
/api/v1/create

Params:
  • file
  • format
 Creates a conversion Job.

Example:
curl --form "file=@test-vid.mp4" --form "format=gif" {{baseUrl}}/api/v1/create
GET /api/v1/status/:jobId

Params:
  • jobId
 Checks status of a previously created job.

Example:
curl {{baseUrl}}/api/v1/status/1403b8ad-ef6a-4de9-9c81-3bc6827e785e
GET /api/v1/resolve/:jobId/:nicename

Params:
  • jobId
  • nicename
 Retrieves the converted output file.

Example:
curl {{baseUrl}}/api/v1/stream/1403b8ad-ef6a-4de9-9c81-3bc6827e785e/test-vid.gif > test-vid.gif
